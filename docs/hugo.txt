# Hugo Documentation Site

## Introduction

The Hugo Documentation Site is the official documentation website for Hugo, the world's fastest static site generator built in Go. This project serves as both comprehensive technical documentation and a living example of Hugo's capabilities, built entirely with Hugo itself. The site provides detailed reference documentation for Hugo's template functions, methods, configuration options, CLI commands, and best practices for building static websites.

The documentation site leverages modern web technologies including Alpine.js for interactive components, Tailwind CSS for styling, and Turbo.js for fast page navigation. It features real-time search powered by Algolia, dark mode support with system preference detection, syntax highlighting for over 150 languages, and an extensive shortcode system for rich content authoring. The project is deployed on Netlify and serves as the primary learning resource for developers using Hugo at https://gohugo.io/.

## APIs and Key Functions

### Algolia Search API Integration

Real-time documentation search powered by Algolia's hosted search service with client-side LRU caching and keyboard shortcuts.

```javascript
// assets/js/alpinejs/data/search.js
export default () => ({
  query: '',
  results: [],
  isOpen: false,
  cache: new LRUCache(10),

  async search() {
    if (this.query.length < 2) {
      this.results = [];
      return;
    }

    const cached = this.cache.get(this.query);
    if (cached) {
      this.results = cached;
      return;
    }

    const response = await fetch(
      `https://D1BPLZHGYQ-dsn.algolia.net/1/indexes/hugodocs/query`,
      {
        method: 'POST',
        headers: {
          'X-Algolia-API-Key': '6df94e1e5d55d258c56f60d974d10314',
          'X-Algolia-Application-Id': 'D1BPLZHGYQ',
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          query: this.query,
          hitsPerPage: 10
        })
      }
    );

    const data = await response.json();
    this.results = data.hits;
    this.cache.set(this.query, this.results);
  },

  // Open search with keyboard shortcut
  init() {
    document.addEventListener('keydown', (e) => {
      if (e.key === '/' && !['INPUT', 'TEXTAREA'].includes(e.target.tagName)) {
        e.preventDefault();
        this.isOpen = true;
      }
    });
  }
});
```

### Table of Contents with Scroll Spy

Automatic table of contents with Intersection Observer-based active section tracking.

```javascript
// assets/js/alpinejs/data/toc.js
export default () => ({
  activeId: '',
  userClicked: false,

  init() {
    const headings = document.querySelectorAll('h2[id], h3[id], h4[id]');

    const observer = new IntersectionObserver((entries) => {
      entries.forEach(entry => {
        if (entry.isIntersecting && !this.userClicked) {
          this.activeId = entry.target.id;
          this.scrollToActive();
        }
      });
    }, {
      rootMargin: '-100px 0px -66%',
      threshold: 0
    });

    headings.forEach(heading => observer.observe(heading));
  },

  scrollToActive() {
    const activeLink = document.querySelector(`a[href="#${this.activeId}"]`);
    if (activeLink) {
      activeLink.scrollIntoViewIfNeeded?.() ||
        activeLink.scrollIntoView({ block: 'nearest' });
    }
  },

  handleClick(id) {
    this.userClicked = true;
    this.activeId = id;
    setTimeout(() => this.userClicked = false, 1000);
  }
});
```

### Dark Mode Store

Global color scheme management with system preference detection and localStorage persistence.

```javascript
// assets/js/alpinejs/stores/nav.js
export default {
  colorScheme: Alpine.$persist('system').as('colorScheme'),

  init() {
    this.applyColorScheme();

    // Watch for system preference changes
    window.matchMedia('(prefers-color-scheme: dark)')
      .addEventListener('change', () => {
        if (this.colorScheme === 'system') {
          this.applyColorScheme();
        }
      });
  },

  applyColorScheme() {
    const isDark = this.colorScheme === 'dark' ||
      (this.colorScheme === 'system' &&
       window.matchMedia('(prefers-color-scheme: dark)').matches);

    document.documentElement.classList.toggle('dark', isDark);
  },

  setColorScheme(scheme) {
    this.colorScheme = scheme;  // 'light', 'dark', or 'system'
    this.applyColorScheme();
  }
};

// Usage in HTML with Alpine.js:
// <button @click="$store.nav.setColorScheme('dark')">Dark Mode</button>
```

### Turbo and Alpine.js Bridge

Seamless integration between Turbo navigation and Alpine.js component lifecycle.

```javascript
// assets/js/helpers/bridgeTurboAndAlpine.js
export function bridgeTurboAndAlpine() {
  // Reinitialize Alpine on Turbo navigation
  document.addEventListener('turbo:load', () => {
    Alpine.initTree(document.body);
  });

  // Cleanup Alpine before cache
  document.addEventListener('turbo:before-cache', () => {
    document.querySelectorAll('[x-data]').forEach(el => {
      Alpine.destroyTree(el);
    });
  });

  // Preserve scroll positions in named containers
  document.addEventListener('turbo:before-render', (event) => {
    const scrollElements = document.querySelectorAll('[data-preserve-scroll-id]');
    scrollElements.forEach(el => {
      const id = el.dataset.preserveScrollId;
      sessionStorage.setItem(`scroll-${id}`, el.scrollTop);
    });
  });

  document.addEventListener('turbo:render', () => {
    const scrollElements = document.querySelectorAll('[data-preserve-scroll-id]');
    scrollElements.forEach(el => {
      const id = el.dataset.preserveScrollId;
      const scrollTop = sessionStorage.getItem(`scroll-${id}`);
      if (scrollTop) el.scrollTop = parseInt(scrollTop);
    });
  });
}
```

### Cascade Configuration

Apply default parameters to content sections and their descendants.

```toml
# hugo.toml - Apply settings to all pages in /news/
[[cascade]]
  [cascade.params]
    hide_in_this_section = true
    show_publish_date = true
  [cascade.target]
    kind = 'page'
    path = '{/news/**}'

# Make all regular pages searchable
[[cascade]]
  [cascade.params]
    searchable = true
  [cascade.target]
    kind = 'page'

# Exclude section pages from search
[[cascade]]
  [cascade.params]
    searchable = false
  [cascade.target]
    kind = '{home,section,taxonomy,term}'
```

## Summary

The Hugo Documentation Site demonstrates best practices for building modern, performant static websites with Hugo. Primary use cases include serving as comprehensive technical documentation with over 1,000 reference pages for template functions, methods, and configuration options; providing interactive learning resources through code examples and tutorials; offering real-time search across the entire documentation corpus; and showcasing Hugo's capabilities including shortcodes, render hooks, asset pipelines, and template system. The site serves thousands of developers daily who are building static websites with Hugo.

Integration patterns demonstrated include Alpine.js for lightweight client-side interactivity without heavy JavaScript frameworks; Algolia for client-side search without backend infrastructure; Tailwind CSS for utility-first responsive design; Turbo.js for fast navigation with preserved state; Netlify for deployment with custom headers and redirects; and data-driven content using YAML/TOML files. The project exemplifies how static site generators can deliver rich, interactive user experiences while maintaining the performance and security benefits of pre-rendered HTML. Developers can study this codebase to learn Hugo template patterns, shortcode development, render hook customization, asset pipeline configuration, and modern frontend integration techniques applicable to any Hugo project.
